
### 引入 Wz

你可以引入整个 Wz，或是根据需要仅引入部分组件。我们先介绍如何引入完整的 WZ

#### 完整引入

在 main.js 中写入以下内容：

```javascript
import Vue from 'vue';
import WzUI from 'wz-ui';
import 'wz-ui/lib/index.css';
import App from './App.vue';

Vue.use(WzUI);

new Vue({
  el: '#app',
  render: h => h(App)
});
```



接下来，如果你只希望引入部分组件，比如 Button 和 Select，那么需要在 main.js 中写入以下内容：

```javascript
import Vue from 'vue';
import { Button, Select } from 'wz-ui';
import App from './App.vue';

Vue.component(Button.name, Button);
/* 或写为
 * Vue.use(Button)
 */

new Vue({
  el: '#app',
  render: h => h(App)
});
```
